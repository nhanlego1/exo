-- SUMMARY --

Tubular lets you set a YouTube video as your page background. It is very easy to configure and use.

-- INSTALLATION --

  1. Download the Tubular jQuery plugin (http://code.google.com/p/jquery-tubular) and extract the file under sites/all/libraries.
  2. Download the swfobject plugin (http://code.google.com/p/swfobject/downloads/list) and extract the file under sites/all/libraries.
  3. Download and enable the Tubular module.
  4. Configure at Administer > Configuration > User interface > Tubular

-- DEPENDENCIES --

  1. Libaries API (http://drupal.org/project/libraries)
